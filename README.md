# Book Explorer

A simple and interactive web application that allows users to explore books and save their favorite ones.  
Built using HTML, Tailwind CSS, and JavaScript.
## Features

- Browse a collection of books
- Add books to favorites
- Remove books from favorites
- Favorites saved using localStorage
- Responsive design (mobile and desktop)
- Mobile navigation menu


## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6 Modules)
- LocalStorage API
- Font Awesome Icons
